# Teacher's Guide: Creating Manual Questions

## Quick Start

When you navigate to the Manual Question Creation page, you can create three types of questions:

---

## üìù Text Question

**Best for:** Essay-type questions, descriptive answers, explanations

### How to Create:
1. Select **"Text Question"** option
2. Choose difficulty level (Easy, Intermediate, or Advanced)
3. Enter your question in the Question field
4. Provide the solution/answer
5. Click "Add Question"

### Example:
- **Question:** "Explain the water cycle in detail."
- **Solution:** "The water cycle consists of evaporation, condensation, precipitation..."

---

## ‚úÖ Multiple Choice Question (MCQ)

**Best for:** Objective questions with single correct answer

### How to Create:
1. Select **"MCQ"** option
2. Choose difficulty level
3. Enter your question
4. Fill in the options (minimum 2 required)
   - Type the text for each option
   - Click the radio button next to the **correct** option
5. Optional: Add more options using "+ Add Option" button
6. Optional: Provide explanation in the Solution field
7. Click "Add Question"

### Tips:
- ‚úÖ Make sure to select the correct answer using the radio button
- ‚úÖ You can have 2 or more options
- ‚úÖ Remove unwanted options using the X button (must keep at least 2)
- ‚úÖ All options must have text before submitting

### Example:
- **Question:** "What is the powerhouse of the cell?"
- **Options:**
  - A. Nucleus
  - B. Mitochondria ‚úì (Correct)
  - C. Ribosome
  - D. Golgi Apparatus
- **Solution:** "Mitochondria are called the powerhouse because they produce ATP..."

---

## üìÑ Fill in the Blanks

**Best for:** Testing specific terms, concepts, or facts

### How to Create:
1. Select **"Fill in the Blanks"** option
2. Choose difficulty level
3. Enter your question using `_____` or `[blank]` to indicate blank spaces
4. Provide correct answers for each blank in order
   - Blank 1 corresponds to the first _____ in your question
   - Blank 2 corresponds to the second _____, and so on
5. Optional: Add more blanks using "+ Add Blank" button if needed
6. Optional: Provide explanation in the Solution field
7. Click "Add Question"

### Tips:
- ‚úÖ Use `_____` (five underscores) or `[blank]` to mark blank spaces
- ‚úÖ Number of blanks should match number of answer fields
- ‚úÖ Provide answers in the order they appear in the question
- ‚úÖ All blanks must have answers before submitting

### Example:
- **Question:** "The capital of France is _____ and it is located on the _____ river."
- **Blank 1:** Paris
- **Blank 2:** Seine
- **Solution:** "Paris is the capital and largest city of France, situated on the Seine..."

## üéØ Difficulty Levels

- **Easy**: Basic recall, simple concepts suitable for beginners
- **Intermediate**: Moderate complexity, requires understanding
- **Advanced**: Complex problems, critical thinking, advanced concepts

## ‚úèÔ∏è Editing Questions

### After Adding:
- Added questions appear in the "Added Questions" list below
- Each question shows:
  - Question number
  - Difficulty badge (color-coded)
  - Question type badge
  - Full content

## üí° Best Practices

### For Text Questions:
- ‚úÖ Be clear and specific
- ‚úÖ Provide comprehensive solutions
- ‚úÖ Use proper grammar and punctuation

### For MCQ:
- ‚úÖ Make options similar in length
- ‚úÖ Avoid "all of the above" or "none of the above" when possible
- ‚úÖ Ensure only one option is clearly correct
- ‚úÖ Provide explanation for why the answer is correct

### For Fill in the Blanks:
- ‚úÖ Don't make blanks too obvious or too obscure
- ‚úÖ Ensure blanks test important concepts
- ‚úÖ Accept common spelling variations (coordinate with backend)
- ‚úÖ Keep the sentence structure clear

---

## ‚ö†Ô∏è Common Mistakes to Avoid

‚ùå **Not selecting correct answer for MCQ** - Always mark the correct option!

‚ùå **Mismatched blanks** - Number of _____ in question should match number of answer fields

‚ùå **Empty options** - All MCQ options must have text

‚ùå **Missing solutions for text questions** - Solutions are required for text questions

‚ùå **Ambiguous questions** - Make sure students understand what you're asking

## üÜò Troubleshooting

### "Add Question" button is disabled:
- Check that question field is not empty
- For MCQ: Ensure all options have text and one is marked as correct
- For Fill in the Blanks: Ensure all blank answers are filled

### Question not appearing in list:
- Check for error messages
- Ensure all required fields are completed
- Try refreshing the page if the issue persists

### Need to edit a question:
- Currently, you need to remove and re-add the question
- Auto-save feature coming soon!
